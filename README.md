# Logeshwaran Portfolio 2026

Next.js portfolio configured for GitHub Pages.

## GitHub UI deployment

1. Upload all files in this folder to the root of your `My_Portfolio` repository.
2. Go to Settings > Pages.
3. Set Source to `GitHub Actions`.
4. Commit/push any file change.
5. Wait for the Actions run to finish.

Live URL:
https://i-am-lucifer.github.io/My_Portfolio/

## Edit these placeholders

- `app/page.jsx`: email, LinkedIn, CV filename, text sections.
- `public/`: add your CV/video/assets later.
